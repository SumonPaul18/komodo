# komodo
Komodo is a web app to provide structure for managing your servers, builds, deployments, and automated procedures.

# Run Komodo on Current Directory
####
    git clone https://github.com/SumonPaul18/komodo.git
    cd komodo
    docker compose up -d
    docker compose ps
